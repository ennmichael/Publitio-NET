Detailed Description

This class is the main interface to the https://publit.io website API. For the most up-to-date and complete documentation, see https://publit.io/docs. Note that this library has a dependency on Newtonsoft.Json, and some of the public methods return instances of Newtonsoft.Json.Linq.JObject. For more information see https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Linq_JObject.htm and https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Linq_JArray.htm.
